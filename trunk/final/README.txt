Molecular Dynamic simple Problem:
--------------------------------

Description:
-----------
Generate trajectories in pdb format for a simple molecular dynamics

Compilation:
-----------
$ svn update
$ make

Execution:
---------
$ ./main > test.pdb

How to view the simulation:
--------------------------

-Linux
$ sudo apt-get install pymol
$ pymol test.pdb
Click to the play buttom

-Other OS (MAC, Windows)
Download pymol http://www.pymol.org/  or vmd http://www.ks.uiuc.edu/Research/vmd/
